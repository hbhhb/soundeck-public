# Soundeck

Soundeck은 사용자가 자신만의 사운드보드를 만들고 관리할 수 있는 웹 애플리케이션입니다. 직관적인 UI를 통해 사운드를 재생하고, 커스텀 사운드 카드를 추가하며, 다양한 오디오 경험을 즐길 수 있습니다.

> 🌐 **[라이브 데모 보기](https://soundeck.vercel.app/)**에서 실제 운영되는 서비스를 확인해보세요.


## ✨ 주요 기능

- **사운드 보드**: 클릭 한 번으로 다양한 효과음과 배경음을 재생하세요.
- **웨이브폼 시각화**: 오디오 파일의 파형을 시각적으로 확인하고 구간을 편집할 수 있습니다.
- **커스텀 사운드**: 드래그 앤 드롭으로 나만의 사운드 파일을 업로드하고 관리하세요.
- **다국어 지원 (i18n)**: 한국어 및 영어를 포함한 다국어 인터페이스를 지원합니다.
- **다크 모드**: 눈이 편안한 다크 모드 테마를 기본으로 제공합니다.
- **반응형 디자인**: 데스크탑과 모바일 환경 모두에 최적화된 UI.
- **클라우드 동기화**: Supabase를 통한 로그인 및 데이터 저장으로 어디서든 내 사운드보드에 접근 가능합니다.

## 🛠 기술 스택

- **Frontend**: [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/), [Shadcn UI](https://ui.shadcn.com/)
- **State Management**: React Context & Hooks
- **Backend & Auth**: [Supabase](https://supabase.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
